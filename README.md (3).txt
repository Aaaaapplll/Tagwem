# ✨ تطبيق تقويم الصلاة الذكي
اسم المبرمج المهندس عبدالله عايد 
تطبيق ويب متكامل لعرض مواقيت الصلاة بناءً على الموقع الجغرافي للمستخدم، بتصميم ذهبي فاخر على خلفية سوداء أنيقة.

## 🌟 المميزات الرئيسية

### ✅ المميزات المنجزة

1. **تحديد الموقع التلقائي**
   - الحصول على الموقع الجغرافي باستخدام GPS المتصفح
   - الرجوع إلى تحديد الموقع عبر IP في حالة عدم توفر GPS
   - عرض اسم المدينة والدولة الحالية
   - إظهار المنطقة الزمنية

2. **عرض مواقيت الصلاة**
   - الفجر
   - الشروق
   - الظهر
   - العصر
   - المغرب
   - العشاء
   - البيانات من Aladhan API (method 4 - أم القرى)

3. **التواريخ المتعددة**
   - التاريخ الميلادي الكامل (باللغة العربية)
   - التاريخ الهجري المحسوب بدقة
   - الساعة الحالية بالوقت الحقيقي

4. **عداد الصلاة القادمة**
   - حساب الوقت المتبقي للصلاة القادمة
   - عرض العداد بالساعات والدقائق والثواني
   - تحديث تلقائي في الوقت الفعلي
   - الانتقال التلقائي للصلاة التالية

5. **التصميم الذهبي الفاخر**
   - خلفية سوداء متدرجة أنيقة
   - لون ذهبي (#D4AF37) للنصوص والحدود
   - تأثيرات بصرية متقدمة (shadows, gradients, animations)
   - تصميم متجاوب يعمل على جميع الأجهزة
   - أيقونات Font Awesome معبرة
   - خط Cairo الأنيق للنصوص العربية

6. **تجربة مستخدم ممتازة**
   - واجهة سهلة الاستخدام وواضحة
   - شاشة تحميل أنيقة
   - رسائل خطأ واضحة
   - تأثيرات تفاعلية عند التمرير
   - تحديث تلقائي للبيانات

## 🚀 كيفية الاستخدام

### البدء السريع

1. **افتح الملف مباشرة:**
   - افتح ملف `index.html` في أي متصفح حديث
   - سيطلب منك المتصفح السماح بتحديد الموقع (اختياري)
   - سيتم عرض مواقيت الصلاة تلقائياً

2. **أو ارفعه على أي استضافة:**
   - ارفع جميع الملفات إلى أي خادم ويب
   - افتح الرابط في المتصفح

### المتطلبات

- متصفح حديث يدعم:
  - JavaScript ES6+
  - Geolocation API (اختياري)
  - Fetch API
- اتصال بالإنترنت لجلب البيانات من API

## 🔧 التقنيات المستخدمة

### Frontend
- **HTML5**: هيكلة الصفحة بشكل دلالي
- **CSS3**: 
  - تصميم متجاوب (Responsive Design)
  - Flexbox و Grid Layout
  - Gradients & Shadows
  - Animations & Transitions
- **JavaScript (Vanilla)**:
  - Geolocation API
  - Fetch API
  - Date manipulation
  - Real-time updates

### المكتبات الخارجية
- **Font Awesome 6.4.0**: للأيقونات
- **Google Fonts (Cairo)**: للخطوط العربية

### خدمات API
1. **Aladhan API** (https://aladhan.com/prayer-times-api)
   - جلب مواقيت الصلاة
   - Method 4: أم القرى، مكة المكرمة

2. **IPApi.co** (https://ipapi.co/)
   - تحديد الموقع عبر IP
   - معلومات الموقع الجغرافي

3. **Nominatim OpenStreetMap** (https://nominatim.openstreetmap.org/)
   - تحويل الإحداثيات إلى أسماء أماكن
   - دعم اللغة العربية

## 📊 نماذج البيانات

### بيانات الموقع
```javascript
{
    latitude: Number,      // خط العرض
    longitude: Number,     // خط الطول
    city: String,         // اسم المدينة
    country: String,      // اسم الدولة
    timezone: String      // المنطقة الزمنية
}
```

### مواقيت الصلاة
```javascript
{
    Fajr: "05:30",       // الفجر
    Sunrise: "06:50",    // الشروق
    Dhuhr: "12:15",      // الظهر
    Asr: "15:30",        // العصر
    Maghrib: "18:00",    // المغرب
    Isha: "19:20"        // العشاء
}
```

### التاريخ الهجري
```javascript
{
    day: Number,          // اليوم (1-30)
    month: Number,        // الشهر (1-12)
    monthName: String,    // اسم الشهر بالعربية
    year: Number          // السنة الهجرية
}
```
